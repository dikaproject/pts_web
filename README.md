### Installation
* composer
  ```sh
  composer install
  ```
* Database Migrate
  ```sh
  php artisan migrate
  ```
  * Database seed
  ```sh
  php artisan db:seed
  ```
* npm
  ```sh
  npm install 
  ```
* npm Run dev in other terminal
  ```sh
  npm run dev
  ```
* start web
  ```sh
  php artisan serve
  ```
